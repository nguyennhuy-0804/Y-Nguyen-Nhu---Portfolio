# Y-Nguyen-Nhu---Portfolio
Data Analysist Portfolio

## [Project 1: Big-Data-Supply-Chain-Analysis-with-Spark-MLlib-and-Spark-SQL]

- This project focused on analyzing key retail metrics, including high- and low-profit products, customer churn, interpurchase intervals, regions with the highest late delivery rates, and RFM-based customer segmentation using Spark SQL. Product grouping and profitability clustering were performed in Python, and sales forecasting was conducted to support strategic planning for a retail store chain.

- Algorithms used: Logistic Regression, Random Forest, K-Means, Prophet

- Link github:  https://github.com/nguyennhuy-0804/Big-Data-Supply-Chain-Analysis-with-Spark-MLlib-and-Spark-SQL.git

## [Project 2: Business Intelligence for Staples Sales Management]

- This project focused on data cleaning, creating interactive dashboards to visualize key metrics, analyzing trends in sales and product profitability across segments, locations, and product categories, and providing actionable business strategies based on the insights.

- Tools & Technologies: Power BI, Excel, R Studio

- Link github:  https://github.com/nguyennhuy-0804/Business-Intelligence-for-Staples-Sales-Management.git
